# Total sales by product
`df.groupby('product')['sales'].sum()`

# Average sales by city
`df.groupby('city')['sales'].mean()`

# Multiple aggregations
```
df.groupby('product').agg({
    'sales': ['sum', 'mean'],
    'quantity': 'sum'
})
```

# Sort values
`df.groupby('product')['sales'].sum().sort_values(ascending=False)`
